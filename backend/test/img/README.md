# 测试图片目录

此目录用于存放 OCR 测试所需的图片文件。

## 支持的图片格式

- `.jpg` / `.jpeg`
- `.png`
- `.bmp`
- `.tiff`
- `.gif`

## 测试图片建议

为了全面测试 OCR 功能，建议准备以下类型的测试图片：

1. **纯文本图片** - 包含清晰的中文或英文文本
2. **混合文本图片** - 包含中文和英文混合的文本
3. **复杂背景图片** - 包含文本但背景较复杂的图片
4. **低质量图片** - 模糊或低分辨率的图片（用于测试错误处理）
5. **空图片** - 不包含文本的图片（用于测试边界情况）

## 注意事项

- 测试图片不会被提交到 Git 仓库（已在 .gitignore 中排除）
- 请确保测试图片不包含敏感信息
- 建议使用公开可用的测试图片或自己创建的示例图片

## 示例测试图片来源

- [Unsplash](https://unsplash.com/) - 免费高质量图片
- [Pexels](https://www.pexels.com/) - 免费图片素材
- 自己使用手机或相机拍摄的包含文本的图片
